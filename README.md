# eGain Knowledge Search Interface

This project is a simplified AI-powered knowledge search interface for customer service agents.

## Features

- Real-time debounced search suggestions
- Article search results with relevance score and preview
- Filtering by category and date range
- Sorting by relevance, date, and popularity
- Article detail modal
- Search history (last 5 searches)
- Loading and error states
- Responsive design
- Mock API (no backend required)

## Tech Stack

- React 18
- TypeScript
- Vite
- Functional components with hooks
- Basic CSS (no external UI framework)

```bash
cd frontend
npm install
npm run dev
```
Open http://localhost:5173 in your broswer.
